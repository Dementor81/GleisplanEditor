Anleitung

Grundlegende Bedienung:

  • Mit der linken Maustaste können Sie Gleise zeichnen
  • Mit der rechten Maustaste können Sie den Bildschirm verschieben
  • Mit dem Mausrad können Sie zoomen
Signale und Weichen:
  • Signale können über das Menü auf der rechten Seite hinzugefügt werden
  • Signale werden durch Ziehen an ein Gleis angeheftet
  • Signale können durch die Alt-Taste für das Gegengleis gedreht werden
  • Weichen werden automatisch erstellt, wenn Sie ein Gleis von einem bestehenden Gleis abzweigen lassen
  • Weichen können durch Klicken umgestellt werden
Züge und Objekte:
  • Züge können über das Menü auf der rechten Seite hinzugefügt werden
  • Züge können mit der Maus verschoben werden
  • Züge können über das Menü gekuppelt und getrennt werden
  • Text und Bahnsteige können über das Objekt-Menü hinzugefügt werden
